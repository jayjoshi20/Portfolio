# AI/ML Engineer Portfolio

This is a personal portfolio website for Jay Joshi, an AI/ML Engineer. It showcases skills, projects, experience, and education in a modern, dark-themed design.

## Features

*   **Hero Section**: Animated background and typing effect.
*   **About Section**: Professional bio and profile image.
*   **Skills Section**: Categorized technical skills.
*   **Experience & Education**: Vertical timeline layout.
*   **Projects Section**: Grid view of high-impact projects with custom images.
*   **Contact Section**: Links to social profiles and email.
*   **Responsive Design**: Works seamlessly on desktop and mobile.

## Tech Stack

*   HTML5
*   CSS3 (Custom properties, Flexbox, Grid)
*   JavaScript (Vanilla)
*   FontAwesome (Icons)
*   Google Fonts (Outfit, Space Mono)

## How to Run Locally

1.  Clone the repository.
2.  Open `index.html` in your browser.

## Deployment

This website is ready to be hosted on GitHub Pages.

1.  Push the code to a GitHub repository.
2.  Go to Settings > Pages.
3.  Select the `main` branch as the source.
4.  Your site will be live at `https://<your-username>.github.io/<repo-name>/`.
